Updating Engimo

Update Engimo clone

Just replace the themes/engimo folder with the latest version of Engimo clone, i.e. delete the old folder and clone again:

rm -rf themes/engimo
git clone --depth 1 https://github.com/MunifTanjim/engimo themes/engimo

Then, commit the changes:

git add themes/engimo
git commit -m "update [theme]: engimo"

Update Engimo submodule

cd themes/engimo
git checkout master
git fetch && git pull
cd ../..
git add themes/engimo
git commit -m "update [theme]: engimo"

To Do After Updating Engimo

After updating Engimo, always check that your site’s config.toml file matches the latest config.toml file format.

A good idea is to double check all the Configuration settings of Engimo.
